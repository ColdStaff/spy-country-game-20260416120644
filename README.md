# Теневая Республика

Многопользовательская браузерная игра про политиков и скрытых шпионов.

## Что уже реализовано

- Комнаты до 20 игроков с кодом, паролем и настройками хоста.
- Роли шпионов по вашему правилу:
  - до 5 игроков: 1 шпион
  - до 8: 2 шпиона
  - до 13: 3 шпиона
  - до 16: 4 шпиона
  - до 20: 5 шпионов
- Цикл раунда:
  - День: голосование по карточкам решений
  - Ночь: приватный канал шпионов (чат + голос)
  - Арест: общее голосование за подозреваемого
  - На следующий день раскрытие: шпион или нет
- Рандом каждый раунд:
  - событие дня
  - сюжетная вставка
  - карточки решений
  - скрытые отложенные последствия
- Показатели страны: экономика, безопасность, благосостояние, доверие, экология, рождаемость.
- Новости по дням для всех игроков.
- Кастомизация внешности и никнейма.
- Голос через микрофон в браузере (WebRTC), включая скрытый ночной канал шпионов.

## Стек

- Node.js + Express
- Socket.IO
- Чистый HTML/CSS/JS (без сборщика)

## Локальный запуск

```bash
npm install
npm start
```

Откройте `http://localhost:3000`.

## Быстрый деплой на бесплатный хостинг

Проект готов под Node-сервис с WebSocket. Подойдут платформы, где поддерживаются долгоживущие процессы и Socket.IO (не serverless).

### Render

1. Залейте проект в GitHub.
2. В Render создайте `Web Service` из репозитория.
3. Укажите:
   - Build Command: `npm install`
   - Start Command: `npm start`
4. Environment:
   - `NODE_VERSION=20` (или выше)
5. После деплоя откройте URL сервиса.

Примечание (актуально на 16 апреля 2026): у Render есть Free web service с лимитами по бесплатным часам/трафику, после исчерпания сервис может быть приостановлен до следующего месяца.

### Railway

1. Подключите GitHub-репозиторий.
2. Создайте новый проект из репозитория.
3. Railway сам определит `npm start`.
4. Откройте выданный домен.

Примечание (актуально на 16 апреля 2026): у Railway есть trial и небольшой бесплатный monthly credit после trial, но с ограничениями по ресурсам.
